# logmarked
A log processor.

## Usage:

    logmarked [OPTIONS,...]

### Options:
  - `c` string
      Configuration file instead of command line parameters. Default empty means using parameters.
  - `i` string
      Logging level.  (default "WARN")
  - `l` string
      Logging output file. Default empty means 'stdout'.
  - `v`
      Display version info.
  - `z`
      Check options.
